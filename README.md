## Project setup
```
npm i
```

Get the Firebase app credentials and update your env:
```
cp .env.example .env
```

### Compiles and hot-reloads for development
```
npm serve
```

### Compiles and minifies for production
```
npm build
```

### Lints and fixes files
```
npm lint
```
